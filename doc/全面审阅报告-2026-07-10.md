## 2. 系统画像

### 2.1 架构（运行时）

```
触发: scheduler | 手动 | webhook | folder_monitor
  → Event → GenerationOrchestrator (TASK_GATE 单飞)
      → ContextBuilder (记忆 / 感知 / 叙事任务卡 / 反完美)
      → PromptBuilder → LLM
      → QAEngine (长度/视角/禁忌/模板/embedding 去重/标题/开场指纹)
```
